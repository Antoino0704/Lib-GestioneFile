Documentazione Libreria GestioneFile

Creare un Oggetto della libreria in scrittura: GestioneFile oggetto = new GestioneFile("path", 'w');

Creare un Oggetto della libreria in append: GestioneFile oggetto = new GestioneFile("path", 'a');

Creare un Oggetto della libreria in lettura: GestioneFile oggetto = new GestioneFile("path", 'r');

Nome API ---Descrizione
Scrivi ---apre il file in modalit√† scrittura, vuole passato come parametro il testo
Leggi ---Legge una riga alla volta e restituisce una stringa
LeggiStampa ---legge l'intero file e lo stampa
chiudi ---chiude il file

esempi:

oggetto.Scrivi("ciao mondo");
var = oggetto.Leggi();
oggetto.LeggiTutto();
oggetto.chiudi();

Includere libreria in Eclipse

1) fai tasto destro sulla parte sinistra del progetto eclipse
2) clicca su Build Path e poi su Configure Build Path...
3)poi clicca su Libraries
4)clicca su Classpath e add External Jars...
5) cerca e selezione il file GestioneFile.jar
6) ricordati di farlo ogni volta per ogni progetto
